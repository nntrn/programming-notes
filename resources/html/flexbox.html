Flexbox utilizes two fundamental types of elements. Their interaction determines how a flex layout will behave:
  • The flex container - the parent element which wraps child elements contained therein. The flex container should have its display property set to flex or inline-flex.
    • The align-items property makes use of this parent-child relationship, as it defines how flex items are laid out along the cross axis on the current line.
  • The flex item(s) - the child element nested within a flex container.
    • Similar to align-items, align-self can be used to define how a single flex item is aligned on the cross axis. It overrides the default value established by align-items.
